# Gallery photos go here

Drop **10 photos** into this folder so they show up in the "Gallery" section of the site.

## Name them exactly like this

```
gallery-01.jpg
gallery-02.jpg
gallery-03.jpg
gallery-04.jpg
gallery-05.jpg
gallery-06.jpg
gallery-07.jpg
gallery-08.jpg
gallery-09.jpg
gallery-10.jpg
```

### Rules (keep it easy)
- Use the **exact names above** (lowercase, `gallery-01` through `gallery-10`, with the leading zero).
- Files must end in **`.jpg`**. If your photos are `.jpeg` or `.png`, just rename the ending to `.jpg` (or re-save them as JPG).
- Square-ish photos look best since the grid crops to squares, but any size works.
- Try to keep each photo **under ~1 MB** so the site stays fast. (You can shrink big photos at [squoosh.app](https://squoosh.app) — no account needed.)

Until a photo is added, the site shows a clean gold placeholder tile in that spot, so nothing looks broken.

**Tip:** put your best/cleanest cuts first (gallery-01, gallery-02) since people look at those first.
